Asynchronous order management for B2B scenario with SAP ERP:

1. Setup the hybris platform in asynchronous scenario using command
./install.sh -r sap_aom_b2b setup

2. Initialize the hybris platform using command 
./install.sh -r sap_aom_b2b initialize

3. Start the hybris server using command
./install.sh -r sap_aom_b2b start

4. Stop the hybris server as well as the datahub service using command
./install.sh -r sap_aom_b2b stop
